Extensão do react para escrever HTML e JS juntos

JSX é uma sintaxe de extensão JavaScript usada no React para escrever facilmente HTML e JavaScript juntos.
Escrever código em React leva muito tempo porque você precisa usar a função React.createElement todas as vezes, mesmo se estiver apenas adicionando um div simples.

Em HTML os elementos podem ficar sem ser fechados
Em JSX todos precisam ser fechados
